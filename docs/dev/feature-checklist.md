# feature-checklist.md

Relicアプリの全機能・UI稼働確認チェックリスト。
実機で動作を確認したら `[x]` にする。

現在の開発位置・セッション開始時に読む文書は [`phases.md`](phases.md) を参照する。

---

## 1. ワークスペース管理

- [ ] 既存フォルダをワークスペースとして開く（「ワークスペースを開く」ボタン）
- [ ] 新規フォルダを作成してワークスペースとして登録する
- [ ] 複数ワークスペースを登録し、一覧から切り替えられる
- [ ] アプリ再起動後に前回のワークスペースが復元される
- [ ] ワークスペースを開いたとき `attachments/` と `templates/` フォルダが自動作成される

---

## 2. ファイル・フォルダ操作

- [ ] Markdownファイルを新規作成できる（ファイル名入力→作成）
- [ ] フォルダを新規作成できる
- [ ] ファイルをリネームできる（RenameBarのインライン入力）
- [ ] フォルダをリネームできる
- [ ] ファイルを複製できる（コマンドパレット「ファイルを複製」）
- [ ] ファイルをゴミ箱に移動できる（コマンドパレット「ファイルを削除」）
- [ ] ファイルをドラッグ&ドロップで別フォルダに移動できる
- [ ] フォルダをドラッグ&ドロップで別フォルダに移動できる
- [ ] ファイル・フォルダをピン留めできる（ピンボタン表示）
- [ ] ピン留めしたファイル・フォルダがツリー上部に表示される
- [ ] ピン留めを解除できる
- [ ] `[[ファイル名]]` のリンクから新規ファイルを作成できる（リンク先不存在時）

---

## 3. ファイルツリービュー

- [ ] ワークスペース内の `.md` ファイルとフォルダがツリー表示される
- [ ] `.md` 拡張子が非表示になっている
- [ ] 非Markdownファイルがツリーに表示されない
- [ ] フォルダを開閉できる
- [ ] アクティブなファイルがツリーでハイライトされる
- [ ] サイドバー幅をドラッグで変更できる（最小180px・最大500px）

---

## 4. エディタ（編集機能）

- [ ] ファイルをクリックするとエディタで開く
- [ ] テキストを入力・編集できる
- [ ] 入力停止後に自動保存される
- [ ] Undo / Redo が動作する（⌘Z / ⌘⇧Z）
- [ ] `[[` を入力するとファイル名補完が表示される
- [ ] ライブプレビュー：カーソルのない行の見出し記号（`#`）が非表示になる
- [ ] ライブプレビュー：カーソルのない行の **太字**・*斜体*・~~打消し~~ が装飾表示される
- [ ] ライブプレビュー：インラインコードが装飾表示される
- [ ] ライブプレビュー：カーソルがある行はMarkdown記法のまま表示される

---

## 5. ツールバー（書式挿入）

- [ ] 太字（**B**）ボタンで `**テキスト**` が挿入される
- [ ] 斜体（*I*）ボタンで `*テキスト*` が挿入される
- [ ] 打ち消し線（S）ボタンで `~~テキスト~~` が挿入される
- [ ] ハイライト（H）ボタンで `==テキスト==` が挿入される
- [ ] 下線（U）ボタンで `<u>テキスト</u>` が挿入される
- [ ] インラインコード（`）ボタンで `` `code` `` が挿入される
- [ ] 見出しドロップダウンから H1〜H6 が挿入される
- [ ] 引用（>）ボタンで `> 引用` が挿入される
- [ ] コードブロックボタンで ` ``` ` が挿入される
- [ ] 水平線（—）ボタンで `---` が挿入される
- [ ] 箇条書きボタンで `- ` が挿入される
- [ ] 番号付きリストボタンで `1. ` が挿入される
- [ ] チェックボックスボタンで `- [ ] ` が挿入される
- [ ] リンクボタンでダイアログが開き `[テキスト](URL)` が挿入される
- [ ] 内部リンクボタンで `[[]]` が挿入される
- [ ] 表ボタンでダイアログが開き、行数・列数を指定してMarkdown表が挿入される

---

## 6. プレビュー

- [ ] ソースモード⇔プレビューモードを切り替えられる
- [ ] 見出し・太字・斜体・リスト・表・引用が正しくレンダリングされる
- [ ] コードブロックに構文ハイライトが適用される
- [ ] `$...$` / `$$...$$` で数式（KaTeX）がレンダリングされる
- [ ] `==ハイライト==` が黄色背景で表示される
- [ ] `<u>テキスト</u>` が下線テキストで表示される
- [ ] `[[ファイル名]]` の内部リンクをクリックするとそのファイルが開く
- [ ] `[[ファイル名#見出し]]` で特定の見出し位置にジャンプする
- [ ] `![[ファイル名]]` で他のMarkdownファイルの内容が埋め込まれる
- [ ] `![[画像ファイル名]]` で attachments フォルダの画像が表示される
- [ ] `[^1]` / `[^1]: 注釈` の脚注が末尾にレンダリングされる
- [ ] プレビュー上のチェックボックスをクリックするとソースが更新される
- [ ] `#tag` タグをクリックするとタグ検索が起動する
