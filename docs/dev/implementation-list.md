# dev/implementation-list.md

ロードマップを日々の実装単位に分解したチェックリスト。

正本は [roadmap.md](roadmap.md)。このファイルは、開発しながら進捗を埋めるための作業リストとして使う。

---

## 運用

- フェーズ順は [roadmap.md](roadmap.md) に従う
- 実装したら該当項目を `[x]` にする
- 新しい判断が必要になったら、ここで決めず [open-questions.md](open-questions.md) に移す
- アプリ本体のコマンドは `app/` で実行する
- フェーズ完了時は `cd app` して `pnpm exec tsc --noEmit` と `pnpm test` を通す
- 必要に応じて `cd app` して `pnpm package` で macOS アプリを生成確認する
