# P6-change-safety.md

Relicの変更管理検討フェーズの正本。

このフェーズでは、Markdownワークスペースの安全な変更管理を検討した。
現在の開発では、現行アプリに残っているローカルMarkdown編集、ファイル管理、検索、リンク、タグ、フロントマターを正とする。

---

## フェーズの目的

- ユーザーファイルを安全に扱う前提を整理する
- ファイル変更時の確認、保存、復元しやすさを検討する
- 書く道具としての安定性を優先する
