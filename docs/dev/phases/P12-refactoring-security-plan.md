# P12-refactoring-security-plan.md

Relicの大規模リファクタリング・安全性強化フェーズの正本。

このフェーズでは、既存機能を維持しながらコード構造、権限境界、入力検証、秘密情報の扱いを段階的に整理した。

---

## フェーズの目的

- 大きくなったUI・メインプロセス処理を責務単位に分ける
- Electron / IPC / ファイル操作の安全境界を明確にする
- 秘密情報、ローカル設定、証明書などをリポジトリに含めない運用を確認する
- 変更単位ごとに型検査と自動テストで挙動維持を確認する

---

## 完了状態

- このフェーズは完了済み
- 今後の実装規約は `docs/dev/conventions.md`、テスト方針は `docs/dev/testing.md` を参照する
