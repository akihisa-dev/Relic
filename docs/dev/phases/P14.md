# P14

Relicの外部連携安全性検討フェーズの正本。

このフェーズでは、外部サービスや認証情報を扱う場合の安全境界を検討した。
現在の開発では、秘密情報をリポジトリ、設定JSON、ワークスペース内ファイル、ログ、エラー詳細へ出さない方針を正とする。

---

## 優先度

1. セキュリティ
2. ユーザー確認と取り消し不能操作の防止
3. ローカルファイルの保護
4. 実装範囲の小ささと検証しやすさ
