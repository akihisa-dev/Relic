# ナビゲーション仕様

Relicのナビゲーションは、左レール、ファイルサイドバー、メインエリアのタブで構成する。

この文書では、各ナビゲーション要素が「何をできるか」を定義する。
画面内の配置は [../design/screens.md](../design/screens.md)、画面遷移やビュー切り替えの流れは [../design/navigation.md](../design/navigation.md) を正とする。

## 左レール

- ファイルボタンはファイルサイドバーを開閉する
- ファイル加工、フロントマター設定、暦設定、設定は、メインエリアのパネルタブとして開く
- チャートは、メインエリアのチャートタブとして開く
- 既に同じパネルタブまたはチャートタブが開いている場合は、そのタブをアクティブにする
- ファイル加工、フロントマター設定、暦設定、設定のパネルタブを開くとき、ファイルサイドバーはファイルビューへ戻す
- ファイルと設定は常に表示する
- ファイル加工、フロントマター設定、暦設定、年表、カレンダーは、対応する機能トグルがオフの場合は表示しない
- 初期状態で表示する入口は、ファイル、カレンダー、設定とする
- ファイル加工、フロントマター設定、暦設定、年表は、初期状態では非表示とする

## ファイルサイドバー

- ファイル・フォルダをツリー表示する
- ワークスペースの作成、登録を行う
- ファイル名、全文、タグ、フロントマターフィールド、正規表現で検索できる
- macOSでは `⌘B`、Windows/Linuxでは `Ctrl+B`、またはボタンで開閉できる
- サイドバー幅はユーザーがドラッグで変更できる。最小幅は180px、最大幅は500pxとする
- 検索コマンドを実行した場合は、ファイルサイドバーを開いて検索入力へフォーカスする

## ピン留め

- ファイル・フォルダ両方をピン留めできる
- ピン留めしたアイテムはファイルサイドバー上部に固定表示する

## タブ

- ファイルタブ、パネルタブ、チャートタブを開ける
- 枚数制限なしで開ける
- タブが多くなると横にスクロールして切り替える
- タブはドラッグ&ドロップで同じペイン内の前後、または分割表示中の別ペインへ移動できる
- タブを閉じた後は、表示履歴ベースで直前に表示していたタブをアクティブにする
- タブ右クリックメニューから、タブを閉じる、他のタブを閉じる、右側のタブを閉じる、すべてのタブを閉じる操作ができる
- タブ右クリックメニューから、タブをピン留め・解除できる
- ピン留めしたタブにはピンアイコンを表示し、タブ列の先頭側に並べる
- 「他のタブを閉じる」「右側のタブを閉じる」「すべてのタブを閉じる」は、ピン留めしたタブを残す
- ファイルタブの右クリックメニューでは、複製、パスコピー、Markdownリンクコピー、OS上の場所表示もできる
- 分割表示中は、タブ右クリックメニューから別ペインで開ける
- タブが0枚になった場合は、新規ファイル作成画面を表示する

## 分割表示

- 左右2ペインのみ対応
- macOSでは `⌘\`、Windows/Linuxでは `Ctrl+\`、またはボタンで分割・解除できる
- 分割表示中は、ペインごとに独立したタブ列を持つ
- ファイルタブ、パネルタブ、チャートタブはいずれもペイン内のタブとして扱う

## 右パネル

- ファイルタブの右側に開閉可能な右パネルを表示する
- 右パネルには、アウトライン / リンク一覧のどちらか一方だけを表示する
- アウトラインでは、見出し（H1〜H6）の一覧を表示し、クリックで該当箇所にジャンプできる
- リンク一覧では、バックリンク・アウトゴーイングリンクを表示する
- リンク一覧の各リンクは、クリックでリンク先または参照元ファイルを開く
- リンク一覧の各リンクは、右クリックメニューから開く、別ペインで開く、Markdownリンクコピー、パスコピー、OS上の場所表示ができる
- 未作成のアウトゴーイングリンクは、リンクとして表示しつつ未作成であることを示す
- アウトゴーイングリンクは最大1,000件まで表示し、上限に達した場合は一部のみ表示していることをリンク一覧に表示する
- バックリンク読み込み中は、読み込み中表示を出す
- 大きいMarkdownとしてソース表示に倒したファイルでは、編集性能を優先してアウトライン、アウトゴーイングリンク、バックリンクの抽出を行わない
- macOSでは `⌘⇧B`、Windows/Linuxでは `Ctrl+Shift+B`、またはボタンで開閉できる
- 右パネル機能トグルがオフの場合は、タブ行アクションの右パネル切り替え操作を表示しない
- 右パネル機能トグルがオフになった場合、表示中の右パネルも閉じる
- 右パネル機能トグルがオフの場合、ショートカットでも右パネルを表示しない
- 右パネル幅はユーザーがドラッグで変更できる。最小幅は220px、最大幅は520pxとする

## ワークスペース

- 新規ワークスペース作成時は、毎回ユーザーが保存場所を選ぶ
- 登録済みワークスペースは左レール下部から切り替える
- ワークスペース切り替え時は、そのワークスペースの前回の状態を復元する
- ただし、別ワークスペースを開く・作成する・切り替える操作では、前ワークスペースの開きタブを閉じる
- 登録済みワークスペースの表示名は、左レール上で直接変更できる
- 登録済みワークスペースは、左レールの右クリックメニューから登録一覧だけ削除できる

## 表示・テーマ

- ダーク／ライトモードはデフォルトでOSの設定に自動追従する
- アプリ設定でダーク／ライトを手動固定することも可能
- アクセントカラーのユーザー指定は持たず、Relic側で定義した統一デザインのみを使用する
- エディタのフォントはアプリが用意したプリセット数種から選択できる

## キーボードショートカット

ショートカットの主修飾キーは、macOSでは `⌘`、Windows/Linuxでは `Ctrl` とする。画面上のショートカット表記も利用中のOSに合わせて出し分ける。

| 操作 | macOS | Windows/Linux |
|------|-------|---------------|
| 検索 | `⌘F` | `Ctrl+F` |
| コマンドパレット | `⌘⇧P` | `Ctrl+Shift+P` |
| クイックスイッチャー | `⌘P` | `Ctrl+P` |
| サイドバー開閉 | `⌘B` | `Ctrl+B` |
| 新規ファイル | `⌘N` | `Ctrl+N` |
| タブを閉じる | `⌘W` | `Ctrl+W` |
| 分割表示 | `⌘\` | `Ctrl+\` |
| 右パネル開閉 | `⌘⇧B` | `Ctrl+Shift+B` |
| タイプライターモード | `⌘⇧T` | `Ctrl+Shift+T` |

## 設定

- 設定パネルでは、テーマ、表示言語、エディタフォント、フォントサイズ、行間、エディタ最大幅、行番号、スペルチェック、フロントマター日付入力順を変更できる
- テーマはシステム追従、ライト、ダークから選ぶ。初期値はシステム追従とする
- 表示言語はシステム追従、英語、日本語から選ぶ。初期値は英語とする
- 機能トグルでは、ファイル加工、フロントマター設定、暦設定、年表、カレンダー、右パネルの表示を切り替えられる
- 初期値は、カレンダーと右パネルをオン、ファイル加工、フロントマター設定、暦設定、年表をオフとする
- 壊れた設定ファイル、配列などオブジェクトではない設定ファイルは、アプリ起動を止めず初期設定として扱う
- 設定保存は安全書き込みで行う
