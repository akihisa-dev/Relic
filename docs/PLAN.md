# Relic — アプリ設計計画書

> 最終更新: 2026-05-01
> ステータス: 計画中

---

## 確定事項

- **アプリ名**: Relic
- **プラットフォーム**: macOS（iOS は将来的に別アプリとして検討）
- **フォーマット**: マークダウン専用（画像なし）
- **テキスト機能**: マークダウンノートアプリとして必要な機能をすべて内蔵する
- **拡張性**: プラグインなし。ユーザーが機能を追加・拡張する仕組みは持たない
- **機能のオンオフ**: ユーザーが設定でトグルできる（どの機能が対象かは未定）
- **GitHub連携**: 任意（なくても使える）。操作レベルはフル（自動同期〜ブランチ管理まで）
- **クラウド同期フォルダ対応**: iCloud Drive / OneDrive / Dropbox など、OS上でローカルフォルダとして見える場所をワークスペースにできる
- **フォルダ**: 任意のローカルフォルダをワークスペースとして登録できる。複数フォルダ対応
- **ライブプレビュー**: アプリの根幹
- **配布**: GitHub Releases（コード署名・App Storeはリリース品質になった段階で検討）
- **開発方法**: Claude Code / Codex を使ったAI駆動開発

## テキスト機能

| 機能 | 説明 |
|------|------|
| フォルダ / ファイル管理 | ネストしたフォルダ構造でノートを整理 |
| マークダウンエディタ | ライブプレビュー + ソースモード切替 |
| 内部リンク `[[...]]` | ノート間をWikiリンクで繋ぐ |
| バックリンク | 自分を参照しているノート一覧を表示 |
| タグ `#tag` | タグで横断的に整理・検索 |
| 全文検索 | フォルダ内の全ノートをリアルタイム検索 |
| テンプレート | 新規ノートにテンプレートを適用 |
| アウトラインパネル | 見出し構造をサイドパネルで表示 |
| コマンドパレット | キーボードでコマンドをすばやく実行 |
| クイックスイッチャー | ファイル名で素早くノートを開く |
| フロントマター (YAML) | ノート冒頭にメタデータを記述 |
| ワードカウント | 文字数・単語数の表示 |
| テーマ | ライト / ダーク |

---

## GitHub連携機能

| 機能 | 説明 |
|------|------|
| GitHub OAuth認証 | アプリ内でGitHubアカウントと連携 |
| リポジトリ接続 | 既存リポジトリをフォルダとして開く |
| 自動プル（起動時） | アプリ起動時に最新を取得 |
| 自動プッシュ（終了時） | アプリ終了時に自動コミット&プッシュ |
| 手動同期ボタン | ワンタップで pull → commit → push |
| コミットメッセージ入力 | 手動同期時にメッセージを書ける |
| コミット履歴 | 過去のコミット一覧を閲覧 |
| 差分表示（Diff） | 変更前後を並べて確認してからコミット |
| ブランチ管理 | ブランチの作成・切り替え |
