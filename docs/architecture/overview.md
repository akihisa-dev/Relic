# architecture/overview.md

Lorebooksのアーキテクチャ全体像。主要コンポーネントの構成と責務を定義する。

---

## 全体構成

Electron アプリは「メインプロセス」と「レンダラープロセス」の2層に分かれる。

```
┌─────────────────────────────────────────────┐
│              Renderer Process               │
│  UI・エディタ・サイドバー・パネル類           │
│  （CodeMirror 6 / 画面描画）                 │
├─────────────────────────────────────────────┤
│                  IPC 通信                   │
│  （レンダラー ↔ メイン間のメッセージング）    │
├─────────────────────────────────────────────┤
│               Main Process                  │
│  ファイル読み書き・Git操作・インデックス管理  │
│  （Node.js / isomorphic-git）               │
└─────────────────────────────────────────────┘
         ↕                    ↕
   iCloud Drive          GitHub (HTTPS)
```

---

## メインプロセスの責務

ファイルシステムやネットワークへのアクセスはすべてメインプロセスが担う。

| コンポーネント | 責務 |
|--------------|------|
| FileSystem | Markdownファイルの読み書き・作成・削除・移動・リネーム |
| GitManager | コミット・プッシュ・プル・ブランチ管理・自動同期・コンフリクト検出 |
| LinkIndex | `[[...]]` 内部リンク・バックリンク・アウトゴーイングリンクの索引を管理 |
| TagIndex | `#tag` とフロントマターの `tags:` フィールドの索引を管理 |
| SearchEngine | 全文検索・ファイル名検索・タグ検索のインデックス管理と検索実行 |
| TemplateManager | テンプレートファイルの管理と新規ファイル作成時の適用 |

---

## レンダラープロセスの責務

画面の描画とユーザー操作の受け付けを担う。

| コンポーネント | 責務 |
|--------------|------|
| Editor | Markdown編集。ライブプレビュー・ソースモード・フォーカスモード・タイプライターモードの切り替えを含む |
| Toolbar | エディタ上部の操作バー。モード切り替えボタン等を配置 |
| FrontmatterForm | ファイル先頭のフロントマターをフォーム形式で表示・編集 |
| Sidebar | ファイルツリー表示・ワークスペース切り替え・ピン留め |
| TabBar | 複数ファイルのタブ管理 |
| SplitView | エディタエリアの左右2ペイン分割表示 |
| SearchPanel | 検索UI・結果一覧表示 |
| OutlinePanel | 見出し一覧表示とジャンプ |
| LinksPanel | バックリンク一覧・アウトゴーイングリンク一覧（ファイル末尾に表示） |
| GitPanel | コミット履歴・Diff表示・ブランチ管理・コンフリクト解決UI |
| StatusBar | 文字数・単語数の表示 |

---

## データの流れ（基本）

1. ユーザーがファイルを開く → レンダラーがメインにIPCで要求
2. メインがファイルを読み取り → レンダラーに内容を返す
3. レンダラーが CodeMirror 6 にセット → 編集可能になる
4. ユーザーが編集 → 一定間隔でメインに保存要求（または手動保存）
5. メインがファイルに書き込み → LinkIndex・TagIndex を更新

---

## ストレージ

| 種別 | 場所 | 内容 |
|------|------|------|
| Markdownファイル | ユーザーが登録した任意のローカルフォルダ内 | ユーザーが作成した `.md` ファイル |
| アプリ設定 | Electronのユーザーデータフォルダ | テーマ・フォントサイズ等の設定値 |
| 検索インデックス | メモリ上（起動時に再構築） | 全文検索・タグ索引 |
