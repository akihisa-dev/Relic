Relicについて
Relicは、ローカルMarkdownを扱うための書き手向けワークスペースです。 ファイルを開き、編集し、リンクでつなげながら文章を育てます。 今見えている状態 - 左にはワークスペースのファイルツリーがあります - 中央にはタブとエディタがあります - 右には見出しまたはリンクの補助パネルがあります 内部リンクは [[今日のメモ]] のように表示されます。 インラインコードは frontmatter のように薄い背景を持ちます。 このメモは現状デザインを検討するためのサンプル本文です。
