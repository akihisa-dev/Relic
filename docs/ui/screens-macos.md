# ui/screens-macos.md

デスクトップ版の画面一覧と各画面の構成。

---

## 基本レイアウト原則

- **左レール + カードサイドバー + タブ式メインエリア**を基本構造にする
- **カードサイドバー**はカードブックの作成・登録、カードツリー、検索、ピン留めを扱う
- **メインエリア**はタブを表示する作業領域とし、カード、年表、カード加工、プロパティ設定、年表設定、設定をタブとして開く
- **右パネル**はカードタブの補助領域として扱い、アウトラインとリンク一覧だけを表示する

---

## 左レール

左端の縦アイコン領域。カードサイドバーの開閉と、メインエリアへ開くパネルタブの入口を持つ。

| 入口 | 動作 |
|------|------|
| カード | カードサイドバーを開閉する |
| カード加工 | メインエリアにカード加工ツールのパネルタブを開く |
| プロパティ | メインエリアにプロパティ設定のパネルタブを開く |
| 年表 | メインエリアに年表タブを開く |
| 年表設定 | メインエリアに年表設定のパネルタブを開く |
| 設定 | メインエリアに設定のパネルタブを開く |

- 各ボタンにはツールチップを表示する
- カード加工とプロパティは、機能トグルがオフの場合は入口を表示しない
- 登録済みカードブックがある場合は、レール下部からカードブックを切り替えられる

---

## カードサイドバー

左レールのカード入口で開閉する可変幅のパネル。

- カードブックの作成、登録
- カードブック内のカードフォルダ・カードのツリー表示
- カード / カードフォルダの作成、リネーム、移動、複製、削除
- ピン留めしたカード / カードフォルダの表示
- カード名、全文、タグ、プロパティフィールド、正規表現による検索
- 検索結果クリックによる該当カードのオープン

サイドバーは `⌘B` とレール上のカードボタンで開閉できる。幅はユーザーがドラッグで変更できる。
登録済みカードブックの切り替えは左レール下部で扱う。

---

## メインエリア

タブ式の作業領域。左右2ペイン分割に対応し、ペインごとに独立したタブ列を持つ。

### タブ種別

| 種別 | 内容 |
|------|------|
| カードタブ | カードを編集する |
| パネルタブ | カード加工、プロパティ設定、年表設定、設定を表示する |
| 年表タブ | `timeline` プロパティをもとに年表を表示する |

### 上部操作バー

- 開いているカードの名前変更
- 開いているカードの移動
- ソースモード切り替え
- 分割表示切り替え
- 右パネルのアウトライン / リンク表示切り替え

カードタブ以外では、カード名変更や移動など、カード固有の操作は表示しない。

---

## カードタブ

カードを表示・編集するタブ。

**固定表示:**
- Markdownツールバー
- エディタ本体
- ステータスバーの文字数・単語数

**表示モード:**
- ライブプレビュー
- ソースモード
- タイプライターモード

プロパティがある場合は、ライブプレビューではカード先頭のYAMLブロックをフォームとして控えめに表示する。ソースモードではフォーム化せず、Markdown内のYAMLとして表示する。

---

## パネルタブ

カード以外の作業画面をメインエリア内のタブとして表示する。

| パネル | 内容 |
|--------|------|
| カード加工 | マージ、分割、タイトル一覧、目次生成を実行する |
| プロパティ設定 | 固定プロパティの説明、カスタムプロパティの入力能力、候補を管理する |
| 年表設定 | 年表についての設定を扱う。P30開始時点では設定項目をまだ置かない |
| 設定 | テーマ、フォント、行間、最大幅、言語、行番号、スペルチェック、機能トグルを管理する |

---

## 右パネル

カードタブの右側に開閉表示される補助領域。

- アウトラインパネル: 見出し一覧とジャンプ
- リンクパネル: バックリンク・アウトゴーイングリンク一覧
- 右パネル内では、アウトライン / リンクのどちらか一方だけを表示する
- 右パネル機能トグルがオフの場合は、上部操作バーの右パネル切り替え操作を表示しない

プロパティの入力能力管理、年表設定、カード加工、設定は右パネルではなく、メインエリアのパネルタブで扱う。

---

## 初回起動時

カードブックが未登録の状態で起動したとき、カードサイドバーに初期設定ビューを表示する。

- 新規カードブックを作成（カードフォルダ名を入力）
- 既存カードフォルダをカードブックとして開く

新規カードブック作成時は、毎回ユーザーが保存場所を選ぶ。
