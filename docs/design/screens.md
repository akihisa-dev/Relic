# design/screens.md

デスクトップ版の画面一覧と各画面の構成。

この文書では、画面内に「何をどこへ置くか」を定義する。
ナビゲーション要素の機能仕様は [../features/navigation.md](../features/navigation.md)、画面遷移やビュー切り替えの流れは [navigation.md](navigation.md) を正とする。

---

## 基本レイアウト原則

- **左レール + ファイルサイドバー + タブ式メインエリア**を基本構造にする
- **ファイルサイドバー**はワークスペースの作成・登録、ファイルツリー、検索、ピン留めを扱う
- **メインエリア**はタブを表示する作業領域とし、Markdownファイル、チャート、ファイル加工、フロントマター設定、設定をタブとして開く
- **右パネル**はMarkdownファイルタブの補助領域として扱い、アウトラインとリンク一覧だけを表示する

---

## 全体レイアウト

```
┌──────────────────────────────────────────────────────┐

│左  │              │  上部操作バー         │           │

```

## 左レール

左端の縦アイコン領域。ファイルサイドバーの開閉と、メインエリアへ開くパネルタブの入口を持つ。

| 入口 | 動作 |
|------|------|
| ファイル | ファイルサイドバーを開閉する |
| ファイル加工 | メインエリアにファイル加工ツールのパネルタブを開く |
| フロントマター | メインエリアにフロントマター設定のパネルタブを開く |
| チャート | メインエリアに年表 / 日付チャートのチャートタブを開く |
| 設定 | メインエリアに設定のパネルタブを開く |

- 各ボタンにはツールチップを表示する
- ファイル加工とフロントマターは、機能トグルがオフの場合は入口を表示しない
- 登録済みワークスペースがある場合は、レール下部からワークスペースを切り替えられる

---

## ファイルサイドバー

左レールのファイル入口で開閉する可変幅のパネル。

- ワークスペースの作成、登録
- ワークスペース内のフォルダ・Markdownファイルのツリー表示
- ファイル / フォルダの作成、リネーム、移動、複製、削除
- ピン留めしたファイル / フォルダの表示
- ファイル名、全文、タグ、フロントマターフィールド、正規表現による検索
- 検索結果クリックによる該当ファイルのオープン

サイドバーは `⌘B` とレール上のファイルボタンで開閉できる。幅はユーザーがドラッグで変更できる。
登録済みワークスペースの切り替えは左レール下部で扱う。

---

## メインエリア

タブ式の作業領域。左右2ペイン分割に対応し、ペインごとに独立したタブ列を持つ。

### タブ種別

| 種別 | 内容 |
|------|------|
| ファイルタブ | Markdownファイルを編集する |
| パネルタブ | ファイル加工、フロントマター設定、設定を表示する |
| チャートタブ | `chronicle` / `plannedDate` / `actualDate` をもとに年表 / 日付チャートを表示する |

### 上部操作バー

- 開いているファイルの名前変更
- 開いているファイルの移動
- ソースモード切り替え
- 分割表示切り替え
- 右パネルのアウトライン / リンク表示切り替え

ファイルタブ以外では、ファイル名変更や移動など、ファイル固有の操作は表示しない。

---

## ファイルタブ

Markdownファイルを表示・編集するタブ。

**固定表示:**
- Markdownツールバー
- エディタ本体
- ステータスバーの文字数・単語数

**表示モード:**
- ライブプレビュー
- ソースモード
- タイプライターモード

フロントマターがある場合は、ライブプレビューではファイル先頭のYAMLブロックをフォームとして控えめに表示する。ソースモードではフォーム化せず、Markdown内のYAMLとして表示する。

---

## パネルタブ

Markdownファイル以外の作業画面をメインエリア内のタブとして表示する。

| パネル | 内容 |
|--------|------|
| ファイル加工 | マージ、分割、タイトル一覧、目次生成を実行する |
| フロントマター設定 | 固定プロパティの説明、カスタムプロパティの入力能力、候補を管理する |
| 設定 | テーマ、フォント、行間、最大幅、言語、行番号、スペルチェック、機能トグルを管理する |

---

## 右パネル

ファイルタブの右側に開閉表示される補助領域。

- アウトラインパネル: 見出し一覧とジャンプ
- リンクパネル: バックリンク・アウトゴーイングリンク一覧
- 右パネル内では、アウトライン / リンクのどちらか一方だけを表示する
- 右パネル機能トグルがオフの場合は、上部操作バーの右パネル切り替え操作を表示しない

フロントマターの入力能力管理やファイル加工、設定は右パネルではなく、メインエリアのパネルタブで扱う。

---

## 初回起動時

ワークスペースが未登録の状態で起動したとき、ファイルサイドバーに初期設定ビューを表示する。

- 新規ワークスペースを作成（フォルダ名を入力）
- 既存フォルダをワークスペースとして開く

新規ワークスペース作成時は、毎回ユーザーが保存場所を選ぶ。
