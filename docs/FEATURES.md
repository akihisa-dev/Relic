# Lorebooks — 機能リスト

ここに残っている機能 = まだ仕様書に落とし込まれていないもの。
仕様書（`docs/spec/`）に書き込まれた機能はここから削除する。

---

## ナビゲーション

| 機能 | 説明 |
|------|------|
| ファイル一覧サイドバー | フォルダ・ファイルをツリー表示 |
| ピン留め | よく使うファイルを上部に固定 |
| タブで複数ファイルを開く | 複数ファイルをタブで切り替え |
| 分割表示 | 2ファイルを左右に並べて表示 |
| アウトラインパネル | 見出し構造をサイドパネルに表示 |

---

## 検索

| 機能 | 説明 |
|------|------|
| 全文検索 | フォルダ内の全ファイルをキーワード検索 |
| ファイル名検索 | ファイル名で絞り込み |
| タグ検索 | タグで絞り込み |
| 正規表現検索 | 正規表現で高度な検索 |
| 検索 & 置換 | 一括置換 |

---

## リンクと参照

| 機能 | 説明 |
|------|------|
| 内部リンク [[...]] | ファイル間をWikiリンクで接続 |
| バックリンク一覧 | そのファイルを参照している一覧を表示 |
| アウトゴーイングリンク一覧 | そのファイルが参照しているリンク一覧 |
| リンクのオートコンプリート | [[を入力するとファイル名を補完 |

---

## 整理 / メタデータ

| 機能 | 説明 |
|------|------|
| タグ #tag | タグで横断的に整理・検索 |
| フロントマター（YAML） | ファイル冒頭にメタデータを記述 |
| テンプレート | 新規ファイルにテンプレートを適用 |

---

## GitHub連携

| 機能 | 説明 |
|------|------|
| OAuth認証 | GitHubアカウントとアプリを接続 |
| リポジトリ接続 | GitHubリポジトリをクローン・ローカルgitフォルダを開く |
| プル | リモートの変更をローカルに取得 |
| コミット + プッシュ | 変更を保存してリモートに送信 |
| 自動プル | 自動でプル（タイミングは未定） |
| 自動プッシュ | 自動でコミット&プッシュ（タイミングは未定） |
| コミットメッセージ入力 | 手動同期時にメッセージを書ける |
| コミット履歴 | 過去のコミット一覧を表示 |
| Diffビュー | 変更前後を並べて確認 |
| ブランチ切り替え | 既存ブランチを切り替え |
| ブランチ作成 | 新しいブランチを作成 |
| コンフリクト通知 | マージ競合が起きたら通知 |

---

## iCloud

| 機能 | 説明 |
|------|------|
| iCloud Drive上のフォルダをフォルダとして開く | iCloudのフォルダをそのままフォルダに |
| iCloud自動同期 | iCloudが自動でデバイス間同期 |

---

## 表示 / テーマ

| 機能 | 説明 |
|------|------|
| システム設定に追従 | OSの外観設定に自動で合わせる |
| フォント選択 | エディタのフォントを変更 |
| カスタムカラーテーマ | 色をカスタマイズ |
| アクセントカラー変更 | UIのアクセントカラーを変更 |

---

## エクスポート

| 機能 | 説明 |
|------|------|
| PDFエクスポート | ファイルをPDFとして書き出す |
| HTMLエクスポート | ファイルをHTMLとして書き出す |
| テキスト / マークダウンエクスポート | ファイルをプレーンテキスト・.mdとして書き出す |
