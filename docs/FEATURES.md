# Lorebooks — 機能リスト

---

## エディタ

| 機能 | 説明 |
|------|------|
| ライブプレビュー（WYSIWYG） | 書きながらリアルタイムにレンダリング |
| ソースモード | マークダウン記法をそのまま表示 |
| フォーカスモード | 周囲を暗くして今の段落に集中 |
| タイプライターモード | カーソルを常に画面中央に固定 |
| ワードカウント / 文字数 | 現在のファイルの文字数・単語数を表示 |
| スペルチェック | システムのスペルチェックと連携 |
| フォントサイズ調整 | エディタの文字サイズを変更 |
| 行間調整 | 行間を広げたり狭めたりできる |
| エディタ最大幅設定 | テキストカラムの幅を制限できる |
| 行番号表示 | 行番号を表示 |

---

## マークダウン記法

| 機能 | 説明 |
|------|------|
| 見出し H1〜H6 | # 記法で見出し |
| 太字 / 斜体 / 打ち消し線 | **bold** *italic* ~~strike~~ |
| コードブロック（シンタックスハイライト） | ```lang でコードを色付き表示 |
| インラインコード | `code` |
| 引用ブロック | > 記法 |
| 箇条書き / 番号付きリスト | - / 1. 記法 |
| チェックボックスリスト | - [ ] タスクリスト |
| 水平線 | --- |
| 表（テーブル） | \| col \| 記法 |
| 脚注 | [^1] 記法 |
| 数式（LaTeX） | $$ 記法で数式をレンダリング |

## ファイル管理

| 機能 | 説明 |
|------|------|
| ファイル作成 / 削除 / リネーム | ファイルの基本操作 |
| フォルダ作成 / 削除 / リネーム | フォルダの基本操作 |
| ファイル移動（ドラッグ&ドロップ） | ファイルをフォルダ間で移動 |
| ファイル複製 | 既存ファイルをコピーして新規作成 |
| ゴミ箱 / 削除確認 | 誤削除を防ぐ |
| 複数フォルダ対応 | 複数の保管庫を切り替えて使う |

## GitHub連携

| 機能 | 説明 |
|------|------|
| OAuth認証 | GitHubアカウントとアプリを接続 |
| リポジトリ接続 | GitHubリポジトリをクローン・ローカルgitフォルダを開く |
| プル | リモートの変更をローカルに取得 |
| コミット + プッシュ | 変更を保存してリモートに送信 |
| 自動プル | 自動でプル（タイミングは未定） |
| 自動プッシュ | 自動でコミット&プッシュ（タイミングは未定） |
| コミットメッセージ入力 | 手動同期時にメッセージを書ける |
| コミット履歴 | 過去のコミット一覧を表示 |
| Diffビュー | 変更前後を並べて確認 |
| ブランチ切り替え | 既存ブランチを切り替え |
| ブランチ作成 | 新しいブランチを作成 |
| コンフリクト通知 | マージ競合が起きたら通知 |
