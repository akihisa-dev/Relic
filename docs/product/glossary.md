# Relic 用語集

Relicの設計・開発・会話で使う主要用語を定義する。

ここから `v0.0.1` として再構成するため、この文書は最小限の用語だけを扱う。
詳細仕様へのリンクは置かず、必要になった時点でユーザーと範囲を確認して追加する。

---

## 基本単位

| 用語 | 英名 | 意味 |
|------|------|------|
| カードブック | Cardbook | Relicに登録するローカルディレクトリ。カードを保管する単位 |
| カード | Card | カードブック内のMarkdownファイル。創作設定の最小単位 |
| カードフォルダ | Card folder | カードブック内でカードを整理するディレクトリ |
| プロパティ | Properties | カード先頭のYAMLブロックに書くメタ情報 |

---

## 編集・閲覧

| 用語 | 英名 | 意味 |
|------|------|------|
| ライブプレビュー | Live preview | Markdownを編集しながら表示整形するエディタ表示 |
| ソースモード | Source mode | Markdown記法をそのまま表示するエディタ表示 |
| タイプライターモード | Typewriter mode | 入力中の行を見失いにくくする表示補助 |
| 内部リンク | Internal link | `[[カード名]]` でカード同士をつなぐリンク |
| バックリンク | Backlinks | 現在のカードを参照しているカードの一覧 |
| アウトゴーイングリンク | Outgoing links | 現在のカードから参照しているカードの一覧 |
| タグ | Tag | カードを横断的に分類するための値 |

---

## 画面

| 用語 | 英名 | 意味 |
|------|------|------|
| 左レール | Left rail | 主要画面や機能へ移動する左端の領域 |
| サイドバー | Sidebar | カードツリーなどを表示する補助領域 |
| メインエリア | Main area | カードやパネルをタブとして表示する中央領域 |
| 右パネル | Right panel | アウトラインやリンク情報を表示する補助領域 |
| タブ | Tab | メインエリアで開く作業単位 |
| パネルタブ | Panel tab | カード以外の設定・ツール画面を表示するタブ |

---

## 機能

| 用語 | 英名 | 意味 |
|------|------|------|
| カード加工ツール | Card tools | カードのマージ、分割、目次生成などを行う機能 |
| プロパティ設定 | Properties settings | プロパティの入力能力や候補を管理する機能 |
| 暦設定 | Calendar settings | 年表や暦に関する設定を扱う画面 |
| コマンドパレット | Command palette | キーボードから操作を呼び出す機能 |
| クイックスイッチャー | Quick switcher | カードを素早く探して開く機能 |
| Timeline | Timeline | カードの時間情報を扱う表示・機能群 |
