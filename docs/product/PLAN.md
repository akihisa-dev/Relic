## 確定事項

- **アプリ名**: Relic
- **プラットフォーム**: macOS / Windows（iOS は、デスクトップ版を自分で継続利用し、外出先・モバイルで使いたい場面が明確になったら別アプリとして検討）
- **フォーマット**: マークダウン中心。Markdown内の添付画像はライブプレビューで表示するが、画像管理アプリにはしない
- **テキスト機能**: マークダウンノートアプリとして必要な機能をすべて内蔵する
- **拡張性**: プラグインなし。ユーザーが機能を追加・拡張する仕組みは将来も原則持たない
- **機能のオンオフ**: ファイル加工ツール、フロントマター設定、右パネルを設定でトグルできる
- **同期・履歴管理**: 現行アプリには持たない。必要な場合はOS・クラウド同期フォルダ・Relic外部のツールで扱う
- **クラウド同期フォルダ対応**: iCloud Drive / OneDrive / Dropbox など、OS上でローカルフォルダとして見える場所をワークスペースにできる
- **フォルダ**: 任意のローカルフォルダをワークスペースとして登録できる。複数フォルダ対応
- **ライブプレビュー**: アプリの根幹
- **配布**: まず自分用ビルド。第三者に配る段階でコード署名と自動アップデートを検討し、一般ユーザー向けに継続提供する価値が見えたら配布方法を判断する
- **料金モデル**: 現時点では設計対象外。まず自分用に作って使い、良ければ収益化を検討する
- **開発方法**: Claude Code / Codex を使ったAI駆動開発

## テキスト機能

| 機能 | 説明 |
|------|------|
| フォルダ / ファイル管理 | ネストしたフォルダ構造でノートを整理 |
| マークダウンエディタ | ライブプレビュー + ソースモード切替 |
| 内部リンク `[[...]]` | ノート間をWikiリンクで繋ぐ |
| バックリンク | 自分を参照しているノート一覧を表示 |
| アウトゴーイングリンク | 自分が参照しているノート一覧を表示 |
| タグ | フロントマター `tags:` で横断的に整理・検索 |
| 全文検索 | フォルダ内の全ノートをリアルタイム検索 |
| フロントマター設定 | 固定プロパティ確認とカスタムプロパティの入力能力管理 |
| アウトラインパネル | 見出し構造をサイドパネルで表示 |
| 右パネル | アウトライン / リンク一覧をファイルタブ横に表示 |
| コマンドパレット | キーボードでコマンドをすばやく実行 |
| クイックスイッチャー | ファイル名で素早くノートを開く |
| フロントマター (YAML) | ノート冒頭にメタデータを記述 |
| 年表 / 日付チャート | `chronicle`、`plannedDate`、`actualDate`、互換用 `date` からチャート表示 |
| ワードカウント | 文字数・単語数の表示 |
| テーマ | ライト / ダーク / システム追従 |
| 添付画像 | Markdown本文に添付した画像をライブプレビューで表示 |
| ファイル加工ツール | マージ・分割・タイトル一覧・目次生成 |
| 設定 | テーマ、言語、エディタ表示、機能トグルを管理 |
