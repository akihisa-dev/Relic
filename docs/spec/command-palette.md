# spec/command-palette.md

コマンドパレットとクイックスイッチャーの仕様。

---

## 目的

キーボード操作で、カード移動とアプリ操作をすばやく実行できるようにする。

---

## コマンドパレット

アプリ内の主要操作を検索して実行するUI。

### 起動

- `⌘⇧P` で開く
- メインエリア上にモーダルとして表示する
- サイドバーの表示状態に関係なく開ける

### 表示内容

- 入力欄
- 入力に一致するコマンド一覧
- 各コマンドの名前
- ショートカットがある場合はショートカット表記

### 対象コマンド

対象カテゴリは以下とする。

| カテゴリ | 例 |
|----------|----|
| カード操作 | 新規カード作成、アクティブカードの場所をカードサイドバーで確認、複製、削除 |
| 表示操作 | サイドバー表示切り替え、右パネル表示切り替え、分割表示 |
| エディタ操作 | タイプライターモード |
| 検索・移動 | カードサイドバーの検索欄へ移動、クイックスイッチャーを開く |
| 設定 | 設定パネルタブを開く |

### 挙動

- 入力に応じて候補を絞り込む
- Enterで選択中のコマンドを実行する
- Escapeで閉じる
- コマンド実行後は原則として閉じる
- 検索コマンドを実行した場合、カードサイドバーが閉じていれば自動で開き、検索欄へフォーカスする
- 設定コマンドを実行した場合、メインエリアに設定パネルタブを開く
- アクティブカード名変更の導線はカードサイドバーを開く。実際のインラインリネームはカードツリーやメインエリア上部操作バーで行う

---

## クイックスイッチャー

カード名でノートをすばやく開くUI。

### 起動

- `⌘P` で開く
- コマンドパレットからも開ける
- メインエリア上にモーダルとして表示する

### 表示内容

- 入力欄
- 入力に一致するカード一覧
- カード名
- 同名カードがある場合はカードフォルダパス

### 対象カード

- 現在のカードブック内の `.md` カードのみを対象にする
- サイドバーに表示しないカード形式は対象外にする

### 挙動

- 入力に応じて候補を絞り込む
- Enterで選択中のカードを開く
- Escapeで閉じる
- 既に開いているカードを選択した場合は、そのタブをアクティブにする
- 開いていないカードを選択した場合は、アクティブなペインの新しいタブで開く
- 分割表示中は、現在アクティブなペインのタブ列に開く

---

## コマンドパレットとクイックスイッチャーの関係

- コマンドパレットは「操作」を探す
- クイックスイッチャーは「カード」を探す
- 全文検索・タグ検索・正規表現検索は [search.md](search.md) の対象とし、この仕様には含めない
