# spec/command-palette.md

コマンドパレットとクイックスイッチャーの仕様。

---

## 目的

キーボード操作で、ファイル移動とアプリ操作をすばやく実行できるようにする。

---

## コマンドパレット

アプリ内の主要操作を検索して実行するUI。

### 起動

- `⌘⇧P` で開く
- メインエリア上にモーダルとして表示する
- サイドバーの表示状態に関係なく開ける

### 表示内容

- 入力欄
- 入力に一致するコマンド一覧
- 各コマンドの名前
- ショートカットがある場合はショートカット表記

### 対象コマンド

初期リリースでは、以下のカテゴリを対象にする。

| カテゴリ | 例 |
|----------|----|
| ファイル操作 | 新規ノート作成、ファイル名変更、複製、削除 |
| 表示操作 | サイドバー表示切り替え、右パネル表示切り替え、分割表示 |
| エディタ操作 | タイプライターモード |
| 検索・移動 | 検索ビューを開く、クイックスイッチャーを開く |
| Git操作 | Gitビューを開く、手動同期、ブランチ切り替え |
| 設定 | 設定ビューを開く |

### 挙動

- 入力に応じて候補を絞り込む
- Enterで選択中のコマンドを実行する
- Escapeで閉じる
- コマンド実行後は原則として閉じる
- 検索・Git・設定ビューを開くコマンドを実行した場合、サイドバーが閉じていれば自動で開く

---

## クイックスイッチャー

ファイル名でノートをすばやく開くUI。

### 起動

- `⌘P` で開く
- コマンドパレットからも開ける
- メインエリア上にモーダルとして表示する

### 表示内容

- 入力欄
- 入力に一致するファイル一覧
- ファイル名
- 同名ファイルがある場合はフォルダパス

### 対象ファイル

- 現在のワークスペース内の `.md` ファイルのみを対象にする
- サイドバーに表示しないファイル形式は対象外にする

### 挙動

- 入力に応じて候補を絞り込む
- Enterで選択中のファイルを開く
- Escapeで閉じる
- 既に開いているファイルを選択した場合は、そのタブをアクティブにする
- 開いていないファイルを選択した場合は、アクティブなペインの新しいタブで開く
- 分割表示中は、現在アクティブなペインのタブ列に開く

---

## コマンドパレットとクイックスイッチャーの関係

- コマンドパレットは「操作」を探す
- クイックスイッチャーは「ファイル」を探す
- 全文検索・タグ検索・正規表現検索は [search.md](search.md) の対象とし、この仕様には含めない
