# spec/markdown.md

マークダウン記法サポートの詳細仕様書。アプリがレンダリング・サポートするマークダウン記法の範囲と挙動を定義する。

**方針：存在するマークダウン記法は可能な限りすべてサポートする。**
標準仕様（CommonMark）に加え、Obsidian互換の拡張記法も含む。

**画像方針：Markdown内の添付画像だけサポートする。** ライブプレビューでは画像として表示するが、Relicは画像管理アプリにはしない。

---

## 標準記法（CommonMark）

| 記法 | 書き方 | 説明 |
|------|--------|------|
| 見出し | `# H1` 〜 `###### H6` | 6段階の見出し |
| 太字 | `**テキスト**` | 太字表示 |
| 斜体 | `*テキスト*` | 斜体表示 |
| 打ち消し線 | `~~テキスト~~` | 打ち消し線 |
| インラインコード | `` `code` `` | インラインのコード表示 |
| コードブロック | ` ```言語名 ` | シンタックスハイライト付きコードブロック |
| 引用ブロック | `> テキスト` | 引用 |
| 箇条書き | `- テキスト` | 順序なしリスト |
| 番号付きリスト | `1. テキスト` | 順序ありリスト |
| チェックボックス | `- [ ] タスク` / `- [x] 完了` | クリックでオン/オフ切り替え可能 |
| 水平線 | `---` | 区切り線 |
| 表（テーブル） | `\| col \|` 記法 | テーブル表示 |
| リンク | `[テキスト](URL)` | 外部リンク |
| 添付画像 | `![alt](attachments/image.png)` | ワークスペース内の添付画像をライブプレビューで表示 |
| 脚注 | `[^1]` / `[^1]: 注釈` | ページ末尾に注釈を表示 |

## 拡張記法（Obsidian互換）

| 記法 | 書き方 | 説明 |
|------|--------|------|
| ハイライト | `==テキスト==` | 黄色のマーカー表示 |
| 下線 | `<u>テキスト</u>` | 下線表示。HTMLタグで唯一サポートする記法。他のHTMLタグはすべてプレーンテキストとして表示する。 |
| 数式（インライン） | `$式$` | LaTeX記法でインライン数式をレンダリング |
| 数式（ブロック） | `$$式$$` | LaTeX記法でブロック数式をレンダリング |
| 内部リンク | `[[ファイル名]]` | アプリ内ノートへのリンク（詳細は links-and-tags.md 参照） |
| 内部リンク（エイリアス） | `[[ファイル名\|表示テキスト]]` | リンクの表示テキストをファイル名と別にする |
| 内部リンク（見出し） | `[[ファイル名#見出し]]` | ファイル内の特定の見出しにリンク |
| ブロック参照 | `[[ファイル名^ブロックID]]` | ファイル内の特定の段落にリンク |
| ファイル埋め込み | `![[ファイル名]]` | 別ファイルの内容をその場に展開表示 |
| タグ | `#タグ名` | タグとして認識（詳細は links-and-tags.md 参照） |
| フロントマター | `---` で囲む YAML | ファイル冒頭のメタデータ（詳細は links-and-tags.md 参照） |

## コードブロックのシンタックスハイライト

- 言語名を指定するとその言語のルールで色付けする
- 対応言語：主要な言語を網羅する（Swift, Python, JavaScript, TypeScript, Bash, JSON, YAML, HTML, CSS など）
- 言語未指定の場合はプレーンテキストとして表示

---

## 添付画像

- Markdown本文中の画像記法 `![alt](path)` をサポートする
- 対象はワークスペース内の添付画像のみとする
- 添付画像は `attachments/` などの専用フォルダに保存する
- ライブプレビューでは画像として表示する
- ソースモードではMarkdown記法として編集できる
- 外部URL画像の表示は初期仕様の対象外とする
- 画像の編集・タグ付け・一覧管理・アルバム管理・画像検索は行わない
- サイドバーと検索の主役は `.md` ファイルのままとする

## 実装ライブラリ

- **シンタックスハイライト**: CodeMirror 6 組み込みのハイライト機能を使用
- **数式レンダリング**: KaTeX（軽量・高速。一般的な数式記号をカバー）

## LaTeX 対応範囲

KaTeX がサポートする標準的なコマンドに準拠する。複雑な LaTeX マクロや特殊パッケージは対象外。
