# spec/markdown.md

マークダウン記法サポートの詳細仕様書。アプリがレンダリング・サポートするマークダウン記法の範囲と挙動を定義する。

**方針：存在するマークダウン記法は可能な限りすべてサポートする。**
標準仕様（CommonMark）に加え、Obsidian互換の拡張記法も含む。

**画像方針：Markdown内の画像記法は編集・表示対象にするが、画像カードそのものは表示しない。** ライブプレビューでは画像プレースホルダーとして扱い、Relicは画像管理アプリにはしない。

---

## 標準記法（CommonMark）

| 記法 | 書き方 | 説明 |
|------|--------|------|
| 見出し | `# H1` 〜 `###### H6` | 6段階の見出し |
| 太字 | `**テキスト**` | 太字表示 |
| 斜体 | `*テキスト*` | 斜体表示 |
| 打ち消し線 | `~~テキスト~~` | 打ち消し線 |
| インラインコード | `` `code` `` | インラインのコード表示 |
| コードブロック | ` ```言語名 ` | シンタックスハイライト付きコードブロック |
| 引用ブロック | `> テキスト` | 引用 |
| 箇条書き | `- テキスト` | 順序なしリスト |
| 番号付きリスト | `1. テキスト` | 順序ありリスト |
| チェックボックス | `- [ ] タスク` / `- [x] 完了` | クリックでオン/オフ切り替え可能 |
| 水平線 | `---` | 区切り線 |
| 表（テーブル） | `\| col \|` 記法 | テーブル表示 |
| リンク | `[テキスト](URL)` | 外部リンク |
| 添付画像 | `![alt](images/image.png)` | 画像記法を画像プレースホルダーとして表示 |
| 脚注 | `[^1]` / `[^1]: 注釈` | ページ末尾に注釈を表示 |

## 拡張記法（Obsidian互換）

| 記法 | 書き方 | 説明 |
|------|--------|------|
| ハイライト | `==テキスト==` | 黄色のマーカー表示 |
| 下線 | `<u>テキスト</u>` | 下線表示。HTMLはDOMPurifyで安全化し、Relicが明示的に保証するHTML由来の装飾は下線のみとする。 |
| 数式（インライン） | `$式$` | LaTeX記法でインライン数式をレンダリング |
| 数式（ブロック） | `$$式$$` | LaTeX記法でブロック数式をレンダリング |
| 内部リンク | `[[カード名]]` | アプリ内ノートへのリンク（詳細は links-and-tags.md 参照） |
| 内部リンク（エイリアス） | `[[カード名\|表示テキスト]]` | リンクの表示テキストをカード名と別にする |
| 内部リンク（見出し） | `[[カード名#見出し]]` | カード内の特定の見出しにリンク |
| ブロック参照 | `[[カード名^ブロックID]]` | カード内の特定の段落にリンク |
| カード埋め込み | `![[カード名]]` | 別カードの内容をその場に展開表示 |
| プロパティ | `---` で囲む YAML | カード冒頭のメタデータ（詳細は frontmatter.md 参照） |

## コードブロックのシンタックスハイライト

- 言語名を指定するとその言語のルールで色付けする
- 対応言語：主要な言語を網羅する（Swift, Python, JavaScript, TypeScript, Bash, JSON, YAML, HTML, CSS など）
- 言語未指定の場合はプレーンテキストとして表示

---

## 添付画像

- Markdown本文中の画像記法 `![alt](path)` をサポートする
- 対象はMarkdown記法そのものとし、カードブック内の相対パス画像も外部URL画像も実画像としては読み込まない
- Relicは画像保存用の専用カードフォルダを自動作成しない
- ライブプレビューでは `alt` またはパスを使った画像プレースホルダーとして表示する
- カーソルまたは選択範囲が画像記法に触れているときは、Markdown記法として編集できる
- 外部URL画像もフェッチせず、画像プレースホルダーとして扱う
- 画像の編集・タグ付け・一覧管理・アルバム管理・画像検索は行わない
- サイドバーと検索の主役は `.md` カードのままとする

---

## カード埋め込み

- `![[カード名]]` で、現在のカードブック内の別カードをその場に表示する
- 埋め込み表示は一段階までとする。埋め込み先の本文にさらに `![[...]]` があっても展開しない
- 大きすぎるカードは全文表示せず、元カードを開いて確認する導線を表示する
- 埋め込み内のリンクをクリックした場合は、通常の内部リンクと同じようにリンク先を開く
- 埋め込み内では直接編集しない。編集する場合は元カードを開く

## 実装ライブラリ

- **編集・ライブプレビューの中心**: CodeMirror 6
- **Markdown解析**: marked
- **脚注拡張**: marked-footnote
- **HTML安全化**: DOMPurify
- **シンタックスハイライト**: highlight.js
- **数式レンダリング**: KaTeX（軽量・高速。一般的な数式記号をカバー）

## LaTeX 対応範囲

KaTeX がサポートする標準的なコマンドに準拠する。複雑な LaTeX マクロや特殊パッケージは対象外。
