# 検索仕様

## 呼び出し方

- ツールバーのボタン、またはショートカットキーで検索を開く

## 検索結果の表示

- 左のサイドバーが検索結果一覧に切り替わる
- ファイル名と一致した行を表示し、クリックで該当箇所にジャンプ・ハイライト

## 対応する検索種別

- 全文検索：フォルダ内の全ファイルをキーワード検索
- ファイル名検索：ファイル名で絞り込み
- タグ検索：タグ（#tag）で絞り込み
- 正規表現検索：正規表現パターンで高度な検索。入力欄の横に「よく使うパターン一覧」の補助メニューを表示し、初心者でも使いやすくする

## 検索＆置換

- 現在開いているファイル内のみの置換と、全ファイル一括置換の両方に対応
