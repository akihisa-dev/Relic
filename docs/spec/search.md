# 検索仕様

## 呼び出し方

- カードサイドバー内の検索欄で検索する
- `⌘F` またはコマンドパレットの検索コマンドで、カードサイドバーを開いて検索欄へフォーカスする
- 検索はカードサイドバーの検索欄に統一し、専用の別画面や別ショートカットは持たない

## 検索結果の表示

- カードサイドバー内に検索結果一覧を表示する
- カード名と一致内容を表示し、クリックで該当カードを開く
- 全文検索・正規表現検索では一致行番号と一致行を表示する
- カード名検索、タグ検索、プロパティ検索では該当カード名と一致理由を表示する

## 対応する検索種別

- 検索対象は現在開いているカードブック内のみ。複数カードブックをまたいだ検索は非対応。
- 全文検索：カードフォルダ内の全カードをキーワード検索。**プロパティも検索対象に含む**
- カード名検索：カード名または `aliases` で絞り込み
- タグ検索：プロパティ `tags:` の値で絞り込み
- プロパティ検索：プロパティ名と検索語句を指定し、対象プロパティの値で絞り込み
- 正規表現検索：正規表現パターンで全文検索する

### 正規表現エラー表示

無効なパターンを入力したとき、検索入力欄にエラー状態のスタイルを付け、「正規表現が正しくありません。」のエラーメッセージを検索結果領域に表示する。

### プロパティ検索

- 検索方法で「プロパティ」を選んだ場合、検索欄とは別にプロパティ名入力欄を表示する
- プロパティ名は、固定プロパティと登録済みカスタムプロパティから選ぶ。カードブック内Markdownに偶然存在する未登録フィールドは候補にしない
- 検索語句は、対象プロパティの既存値候補がある場合は入力候補として表示する
