# 検索仕様

## 呼び出し方

- ファイルサイドバー内の検索欄で検索する
- `⌘F` またはコマンドパレットの検索コマンドで、ファイルサイドバーを開いて検索欄へフォーカスする

## 検索結果の表示

- ファイルサイドバー内に検索結果一覧を表示する
- ファイル名と**一致行のみ**を表示し、クリックで該当箇所にジャンプ・ハイライト（前後の文脈行は表示しない）

## 対応する検索種別

- 検索対象は現在開いているワークスペース内のみ。複数ワークスペースをまたいだ検索は非対応。
- 全文検索：フォルダ内の全ファイルをキーワード検索。**フロントマターも検索対象に含む**
- ファイル名検索：ファイル名で絞り込み
- タグ検索：フロントマター `tags:` の値で絞り込み
- 正規表現検索：正規表現パターンで高度な検索。入力欄の横に「よく使うパターン一覧」の補助メニューを表示し、初心者でも使いやすくする

### 正規表現エラー表示

無効なパターンを入力したとき、入力欄を赤くハイライトし、「正規表現が正しくありません」のエラーメッセージを表示する。

### よく使うパターン一覧

| 名称 | パターン |
|------|---------|
| 行頭が見出し | `^#+ ` |
| URLを含む行 | `https?://` |
| 日付形式 | `\d{4}-\d{2}-\d{2}` |
| タグ記法 | `#\w+` |

## 検索＆置換

- 現在開いているファイル内のみの置換と、全ファイル一括置換の両方に対応
- **全ファイル一括置換**はプレビュー画面あり：対象ファイル数・箇所数と変更一覧を確認してから実行する
