# spec/editor.md

エディタ機能の詳細仕様書。テキスト編集に関わるモード・表示・操作の振る舞いを定義する。

---

## モード

| モード | 説明 |
|--------|------|
| ライブプレビュー | デフォルト。マークダウン記法をリアルタイムにレンダリングして表示する |
| ソースモード | マークダウン記法をそのまま表示する。レンダリングなし |
| フォーカスモード | 現在のカーソルがある段落以外を薄暗く表示する。他のモードと併用可能 |
| タイプライターモード | カーソル行を常に画面中央に固定してスクロールする。他のモードと併用可能 |

- **デフォルト**：ライブプレビュー
- ライブプレビュー ↔ ソースモードはツールバーのボタンで切り替え
- フォーカスモード・タイプライターモードはそれぞれ独立してオン/オフ切り替え可能

---

## 表示設定

| 設定項目 | デフォルト | 説明 |
|----------|-----------|------|
| フォントサイズ | 16px | エディタの文字サイズを変更できる |
| 行間 | 1.7 | 行の間隔を広げたり狭めたりできる |
| エディタ最大幅 | 未定 | プリセットから選んで設定する（詳細は下記） |
| 行番号表示 | オフ | 設定でオン/オフを切り替えられる |
| スペルチェック | オン | OSのスペルチェックと連携。誤字に赤下線を表示。設定でオフにできる |

### エディタ最大幅

テキストカラムの最大幅をプリセットから選んで設定できる。ウィンドウを広げてもテキストが伸びすぎず読みやすい幅に収まる。

| プリセット名 | 幅 | 1行の目安文字数（日本語） |
|---|---|---|
| 狭め | 550px | 約30文字 |
| 標準 | 660px | 約36文字 |
| 広め | 800px | 約44文字 |
| 制限なし | — | ウィンドウ全幅 |

- **デフォルト**：標準（660px）

---

## ステータスバー

画面下部にステータスバーを表示する。以下の情報を常時表示する：

- 現在のファイルの文字数
- 現在のファイルの単語数

---

## ツールバーボタン

エディタ上部のツールバーに以下のボタンを配置する。ボタンを押すと選択中のテキストに対応するMarkdown記法を適用する（選択なしの場合はカーソル位置に記法を挿入する）。

### 書式系

| ボタン | 適用される記法 |
|--------|--------------|
| 太字 | `**テキスト**` |
| 斜体 | `*テキスト*` |
| 打ち消し線 | `~~テキスト~~` |
| ハイライト | `==テキスト==` |
| 下線 | `<u>テキスト</u>` |
| インラインコード | `` `テキスト` `` |

### ブロック系

| ボタン | 適用される記法 |
|--------|--------------|
| 見出し | H1〜H6をドロップダウンで選択して適用 |
| 引用 | `> テキスト` |
| コードブロック | ` ```言語名 ` で囲む |
| 水平線 | `---` を挿入 |

### リスト系

| ボタン | 適用される記法 |
|--------|--------------|
| 箇条書き | `- テキスト` |
| 番号付きリスト | `1. テキスト` |
| チェックボックス | `- [ ] テキスト` |

### 挿入系

| ボタン | 動作 |
|--------|------|
| リンク | `[テキスト](URL)` を挿入。URLの入力ダイアログを表示する |
| 内部リンク | `[[` を挿入し、ファイル名のオートコンプリートを起動する |
| 表 | 行数・列数を指定するダイアログを表示し、テーブルのMarkdownを挿入する |
