# ナビゲーション仕様

Relicのナビゲーションは、左レール、カードサイドバー、メインエリアのタブで構成する。

## 左レール

- カードボタンはカードサイドバーを開閉する
- カード加工、プロパティ設定、設定は、メインエリアのパネルタブとして開く
- Chronicleは、メインエリアのChronicleタブとして開く
- 既に同じパネルタブまたはChronicleタブが開いている場合は、そのタブをアクティブにする
- カード加工とプロパティ設定は、対応する機能トグルがオフの場合は表示しない

## カードサイドバー

- カード・カードフォルダをツリー表示する
- カードブックの作成、登録を行う
- カード名、全文、タグ、プロパティフィールド、正規表現で検索できる
- ボタンまたは `⌘B` で開閉できる
- サイドバー幅はユーザーがドラッグで変更できる
- 検索コマンドを実行した場合は、カードサイドバーを開いて検索入力へフォーカスする

## ピン留め

- カード・カードフォルダ両方をピン留めできる
- ピン留めしたアイテムはカードサイドバー上部に固定表示する

## タブ

- カードタブ、パネルタブ、Chronicleタブを開ける
- 枚数制限なしで開ける
- タブが多くなると横にスクロールして切り替える
- タブを閉じた後は、表示履歴ベースで直前に表示していたタブをアクティブにする
- タブが0枚になった場合は、新規カード作成画面を表示する

## 分割表示

- 左右2ペインのみ対応
- `⌘\` またはボタンで分割・解除できる
- 分割表示中は、ペインごとに独立したタブ列を持つ
- カードタブ、パネルタブ、Chronicleタブはいずれもペイン内のタブとして扱う

## 右パネル

- カードタブの右側に開閉可能な右パネルを表示する
- 右パネルには、アウトライン / リンク一覧のどちらか一方だけを表示する
- アウトラインでは、見出し（H1〜H6）の一覧を表示し、クリックで該当箇所にジャンプできる
- リンク一覧では、バックリンク・アウトゴーイングリンクを表示する
- `⌘⇧B` またはボタンで開閉できる

## カードブック

- 新規カードブック作成時は、毎回ユーザーが保存場所を選ぶ
- 登録済みカードブックは左レール下部から切り替える
- カードブック切り替え時は、そのカードブックの前回の状態を復元する

## 表示・テーマ

- ダーク／ライトモードはデフォルトでOSの設定に自動追従する
- アプリ設定でダーク／ライトを手動固定することも可能
- アクセントカラーのユーザー指定は持たず、Relic側で定義した統一デザインのみを使用する
- エディタのフォントはアプリが用意したプリセット数種から選択できる

## キーボードショートカット

| 操作 | ショートカット |
|------|----------------|
| 検索 | `⌘F` |
| コマンドパレット | `⌘⇧P` |
| クイックスイッチャー | `⌘P` |
| サイドバー開閉 | `⌘B` |
| 新規カード | `⌘N` |
| タブを閉じる | `⌘W` |
| 分割表示 | `⌘\` |
| 右パネル開閉 | `⌘⇧B` |
| タイプライターモード | `⌘⇧T` |
