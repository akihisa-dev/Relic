# ナビゲーション仕様

## サイドバー

- ファイル・フォルダをツリー表示（階層的に展開・折りたたみ可能）
- ボタンまたは `⌘B` で開閉できる
- サイドバー左端に縦アイコンナビを置き、ファイルツリー / 検索 / Git / 設定を切り替える
- サイドバー幅はユーザーがドラッグで変更できる
- サイドバーを閉じた状態で検索・Git・設定を開いた場合は、サイドバーを自動で開いて対象ビューを表示する

## ピン留め

- ファイル・フォルダ両方をピン留めできる
- ピン留めしたアイテムはサイドバー上部に固定表示

## タブ

- 枚数制限なしで開ける
- タブが多くなると横にスクロールして切り替える
- タブを閉じた後は、表示履歴ベースで直前に表示していたタブをアクティブにする
- タブが0枚になった場合は、新規ノート作成画面を表示する

## 分割表示

- 左右2ペインのみ対応
- `⌘\` またはボタンで分割・解除できる
- 分割表示中は、ペインごとに独立したタブ列を持つ

## 右パネル

- エディタ右側に開閉可能な右パネルを表示する
- 右パネルには、アウトライン / リンク一覧のどちらか一方だけを表示する
- アウトラインでは、見出し（H1〜H6）の一覧を表示し、クリックで該当箇所にジャンプできる
- リンク一覧では、バックリンク・アウトゴーイングリンクを表示する
- `⌘⇧B` またはボタンで開閉できる

## ワークスペース

- 新規ワークスペース作成時は、毎回ユーザーが保存場所を選ぶ
- ワークスペース切り替え時は、そのワークスペースの前回の状態（タブ・分割・右パネル）を復元する

## 表示・テーマ

- ダーク／ライトモードはデフォルトでOSの設定に自動追従する
- アプリ設定でダーク／ライトを手動固定することも可能
- アクセントカラーのユーザー指定は持たず、Relic側で定義した統一デザインのみを使用する
- エディタのフォントはアプリが用意したプリセット数種から選択できる

## キーボードショートカット

| 操作 | ショートカット |
|------|----------------|
| 検索 | `⌘F` |
| ワークスペース全体検索 | `⌘⇧F` |
| コマンドパレット | `⌘⇧P` |
| クイックスイッチャー | `⌘P` |
| サイドバー開閉 | `⌘B` |
| 新規ノート | `⌘N` |
| タブを閉じる | `⌘W` |
| 分割表示 | `⌘\` |
| 右パネル開閉 | `⌘⇧B` |
