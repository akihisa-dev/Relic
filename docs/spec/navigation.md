# ナビゲーション仕様

## サイドバー

- ファイル・フォルダをツリー表示（階層的に展開・折りたたみ可能）
- ボタン（またはショートカット）で開閉できる

## ピン留め

- ファイル・フォルダ両方をピン留めできる
- ピン留めしたアイテムはサイドバー上部に固定表示

## タブ

- 枚数制限なしで開ける
- タブが多くなると横にスクロールして切り替える

## 分割表示

- 左右2ペインのみ対応
- ショートカットやボタンで分割・解除できる

## アウトラインパネル

- エディタ右側にパネルを表示
- 見出し（H1〜H6）の一覧を表示し、クリックで該当箇所にジャンプ
- ボタンで開閉できる
