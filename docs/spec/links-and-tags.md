# spec/links-and-tags.md

このカードは内部リンク・バックリンク・タグ機能の詳細仕様書です。
`[[...]]` リンク・バックリンク一覧・プロパティ `tags:` による整理・オートコンプリートなど、
ノート間の接続に関わる機能の振る舞いをここに定義します。

---

## 内部リンク `[[...]]`

- リンクの対象は現在開いているカードブック内のカードのみ。複数カードブック間のリンクは非対応。
- `[[カード名]]` 記法でカード間を接続する内部リンクを作成する
- リンク先カードが存在しない場合：
  - リンクとして表示しつつ、色や下線などのスタイルで「未作成」であることを示す
  - クリックすると自動で新規カードを作成し、そのカードを開く（作成場所：リンクが書かれているカードと同じカードフォルダ）
- `[[` を入力した時点で、カード名のオートコンプリートをカーソル直下にドロップダウンリストで表示する
- パス指定 `[[カードフォルダ名/カード名]]` の書き方をサポートする（手打ち・補完どちらも可）
- 同じカード名が複数カードフォルダに存在する場合、補完候補に `カードフォルダ名/カード名` の形式で両方を表示し、ユーザーが選択する。選択したリンクはパス付き `[[カードフォルダ名/カード名]]` で挿入される

### 対応する記法バリエーション

| 記法 | 動作 |
|------|------|
| `[[カード名]]` | カードを開く |
| `[[カード名\|表示テキスト]]` | 表示テキストを変えてカードを開く |
| `[[カード名#見出し]]` | カードを開き、指定した見出しの位置にスクロール |
| `[[カード名^ブロックID]]` | カードを開き、指定したブロックの位置にスクロール |
| `![[カード名]]` | カードの内容をその場に展開して表示（埋め込み） |

### ブロック参照について

- ツールバーのブロックID挿入ボタンで、カーソル位置の段落末尾に `^abc123` のようなIDを自動生成して挿入できる
- テキストを複数段落にまたがって選択している場合、選択範囲内の各段落末尾にIDを挿入できる
- 既に段落末尾にブロックIDがある場合は重複挿入しない
- ブロックIDは手入力してもよい
- 既存の手書きIDはリンク解決・リネーム追跡の対象として読み取る

### カード埋め込みについて

- `![[カード名]]` は一段階だけ展開表示する
- 埋め込み先の本文にさらに `![[...]]` があっても、その内側の埋め込みは展開しない
- 大きすぎるカードは全文表示せず、元カードを開いて確認する導線を表示する
- 埋め込み内リンクは通常の内部リンクと同じように開く
- 埋め込み内では直接編集せず、編集する場合は元カードを開く

## バックリンク・アウトゴーイングリンク

- **バックリンク一覧**：そのカードを参照しているカードの一覧
- **アウトゴーイングリンク一覧**：そのカードが参照しているリンクの一覧
- バックリンク・アウトゴーイングリンクはナビゲーション補助として位置づける
- 両一覧ともに、右パネル（アウトラインパネルと同じ領域）に表示する
- エクスポート・印刷の対象には含まれない

---

## タグ

- タグはプロパティの固定プロパティ `tags:` に記述する
- 本文中の `#タグ名` 記法はタグとして扱わない
- 本文中で概念・人物・資料などを参照する場合は、インラインリンク `[[...]]` を使う
- タグを使って横断的な検索・絞り込みが可能
- **階層タグをサポート**：`親/子` の形式で `/` によるネストが可能（例：`キャラ/主人公`）
- **日本語タグをサポート**：日本語・漢字・ひらがな・カタカナを含むタグ名が使用可能
- タグ入力時は、カードブック内の既存 `tags:` から候補を出し、表記揺れを抑える

## プロパティ（YAML）

プロパティの詳細仕様は [spec/frontmatter.md](frontmatter.md) を参照。

- カード先頭の `---` ブロックをプロパティとして認識する
- `tags:` フィールドはRelicの固定プロパティとしてタグ検索・タグ一覧に使われる（詳細は frontmatter.md 参照）

---

## プロパティテンプレート

- プロパティテンプレートは、複数の能力付きフィールド名をまとめるアプリ設定上の保存済みセットとして扱う
- 現行UIでは、テンプレート管理画面やテンプレート適用操作は表示しない
- テンプレートはアプリ設定として保存し、カードブック内の `templates/` カードフォルダや `.md` テンプレートカードには依存しない
- 新規カード作成時に本文テンプレートを選択する機能は、現行アプリには持たない
- 詳細は [frontmatter.md](frontmatter.md) を参照する
