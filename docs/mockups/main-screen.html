エディタ本体

ここはMarkdownファイルの本文を表示・編集する領域です。 この領域を「エディタ本体」と呼びます。

本文内の 内部リンク や見出し、段落、引用などがここに表示されます。 右側の補助領域はペインではなく「右パネル」です。

見出し

エディタを左右に分割した場合、それぞれの表示枠を「ペイン」と呼びます。 このモックで右端にある補助領域は「右パネル」です。

引用ブロック

下部には「ステータスバー」があり、現在のファイル状態や文字数などを表示します。
