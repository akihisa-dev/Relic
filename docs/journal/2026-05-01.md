## 決定したこと

- **プラットフォーム**: macOS 専念。iOS は将来的に別アプリとして検討
- **技術スタック**: Electron + TypeScript
  - マークダウンエディタ系のライブラリ（CodeMirrorなど）が豊富
  - AI駆動開発との相性が良い
  - Swift/SwiftUIはiOS含む両対応が難しく、エディタ実装が鬼門だった
- **配布**: GitHub Releases。コード署名・App Storeはリリース品質になった段階で検討
- iOSはPWAもReact Nativeも一長一短があり、今は考えない
- Mac App Storeの権威性は低い（ChromeやVSCodeも非掲載）ので、GitHub配布で十分
