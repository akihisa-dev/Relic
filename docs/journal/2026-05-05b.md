フェーズ6後半の GitHub 連携を進めた。

### GitHub OAuth / Keychain

- ブラウザOAuthによるGitHubアカウント接続を実装
  - `RELIC_GITHUB_CLIENT_ID` / `RELIC_GITHUB_CLIENT_SECRET` を使う構成
  - `127.0.0.1` の一時HTTPサーバーでOAuth callbackを受け取る
  - GitHubユーザー名とscopeを取得して接続状態として表示
- OAuthで取得したアクセストークンをmacOS Keychainへ保存
  - 通常の設定JSONにはアクセストークンを保存しない
  - GitHub接続解除時にKeychainから削除する
- GitサイドバーにGitHub接続状態、接続、解除UIを追加

### GitHub remote / push / pull

- 既存ローカルGitリポジトリにGitHub remoteを後付け接続する処理を実装
  - GitHub HTTPS URLを `origin` として登録
  - `.git` 付きURLへ正規化
  - remote一覧をGitサイドバーに表示
- Keychainに保存したOAuthトークンを使い、isomorphic-git経由で現在ブランチの `Pull` / `Push` を実行できる入口を追加
- GitタグをGitHubへpushする処理とUIを追加
- IPC / preload / shared型定義を追加
