# Code of Conduct

Relic aims to be a respectful and practical open source project.

## Expected Behavior

- Be respectful and constructive.
- Discuss technical disagreements with concrete reasons.
- Assume public project spaces may be read by many people.
- Avoid sharing private files, credentials, tokens, or personal data.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing another person's private information.
- Posting secrets, credentials, or exploit details in public issues.
- Repeated off-topic or disruptive behavior.

## Reporting

For ordinary project issues, use GitHub Issues.

For vulnerabilities, leaked secrets, or sensitive security details, do not post the details publicly. Use GitHub Security Advisories or another private contact path offered by the repository maintainers.

Maintainers may edit, hide, lock, or remove content and may restrict participation when needed to keep the project safe and focused.
