# Lorebooks

書き手のための、Git付きマークダウンエディタ。

GitHubは強力だが、技術者向けに作られている。Obsidianは高機能だが、バージョン管理の恩恵を受けられない。Lorebooksはその空白を埋める——コードを書かない人間が、自分のテキストの歴史を管理できるアプリ。

> ステータス：開発中

---

## 対象ユーザー

- オリジナルの世界観や設定をこつこつ作っている人
- TRPGのシナリオや世界観を整理したい人
- キャラクターや歴史をノートに書き溜めている人

---

## 主な機能

- マークダウンエディタ（ライブプレビュー / ソースモード）
- フォルダ・ファイル管理（ネスト対応）
- 内部リンク `[[...]]` とバックリンク
- タグ `#tag` と全文検索
- GitHub連携（コミット・プッシュ・差分表示・ブランチ管理）
- コマンドパレット / クイックスイッチャー

---

## 技術スタック

- **フレームワーク**: Electron
- **言語**: TypeScript
- **エディタエンジン**: CodeMirror 6
- **Git**: simple-git
